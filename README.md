# 💼 ERP System für Buchhaltung & Mitarbeiterverwaltung

## 🚀 Ziel
Entwicklung einer ERP-Software zur Unterstützung einer Steuerkanzlei, inklusive Buchhaltung, Mandantenverwaltung und HR-Prozessen.

## 📊 Funktionen

### 🧾 Buchhaltung
- Schnellerfassung von Buchungen (SKR04)
- Verarbeitung von Buchhaltungsdaten
- Unterstützung von Accounting-Prozessen

### 👥 Mandantenverwaltung
- Verwaltung von Mandanten und Stammdaten
- Strukturierung von Geschäftsdaten

### 🕒 Mitarbeiter-System
- Elektronische Zeiterfassung (Stempeln)
- Verwaltung von Urlaubsanträgen
- HR-Prozessunterstützung

## 🧠 Technologien
- Python (Flask / Django)
- SQL (PostgreSQL / MSSQL)
- Datenmodellierung
- Webentwicklung

## 📈 Mehrwert
- Digitalisierung einer Steuerkanzlei
- Automatisierung von Buchhaltungsprozessen
- Zentrale Verwaltung von Geschäfts- und Personaldaten
